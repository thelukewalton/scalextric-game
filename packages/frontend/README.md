# Flutter front end for the RFID Scalextric Game

## Setup and Running

### Prerequisites

- Flutter SDK: [Install Flutter](https://flutter.dev/docs/get-started/install)
- Dart SDK: Comes with Flutter
- An IDE or text editor: [VS Code](https://code.visualstudio.com/) or [Android Studio](https://developer.android.com/studio)
  > Note: Android studio is required to build the app for Android devices. VS Code is optional.

### Installation

1. Get the dependencies:
   ```sh
   flutter pub get
   ```

### Running the App

1. Connect a device
2. Run the app:
   ```sh
   flutter run
   ```

### Building for Release

1. Build the app for Android:

   ```sh
   flutter build apk --no-tree-shake-icons
   ```

For more details, refer to the [official Flutter documentation](https://flutter.dev/docs).
